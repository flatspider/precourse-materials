### CSS QUESTIONS

## What does CSS stand for? What is it used for?

- CSS stands for cascading style sheets. It was invented to add layout styling for web pages. Prior to its invention, there was no way to manage the look and feel of a site.

## What is a CSS pre-processor? Can you give some examples?

- A CSS pre-processor generates CSS from the pre-processors unique syntax. For example Sass is a CSS pre-processor. Sass has syntax improvements (such as nesting and declarations) that allow for more organization and clarity than pure CSS. There are also Less and Stylus CSS pre-processors.

## What is a CSS framework? Can you give some examples?

- A CSS framework is a library of CSS components. Bootstrap and tailwind are famous frameworks that allow for rapidly building style components. For example, this allows you to use <html class="accordion" id="accordion-example"> and use the component from Bootstrap.
